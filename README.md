
# Särmä (ios app)

Särmä (meaning sharp/ready in finnish army slang) is ios mobile app based on swiftui utilizing swiftdata. The is build for finnish conscripts currently in service or going in service. It serves as companion app providing up to date knowledge for people going into service and as TJ ("tänään jäljellä") countdown for those already in the army. 


## Features

- Modern ios26 design
- menu that has <2s load time
- Budgeting and income tracking tool
- Cooper test track view
- Packing list for new conscripts
- Leave and holiday planing view
- Widget support 3 sized (small, medium, large)
- No ads




## FAQ

#### Is the app on app store ?

As of January 2026 the app is on apple app store with the name Särmä

### Is the app free ?

The app and all core features except widgets are free.

### When should I download this app ?

The app content is designed to be completed 4-12 weeks before service startdate. For example someone starting in july might download the app in April.
